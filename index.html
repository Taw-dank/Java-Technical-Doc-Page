Introduction

Java is an high-level, object-oriented programming language. It is a write one, read anywhere meaning compiled Java code can be run on any device with a Java Virtual Machine (JVM).

Java is widely used in server-side development to build secure web aplications and services.

  • Web applications
  • Apps
  • Enterprise Software

Java is known for its security, scalability and memory management.

What you should know

You should be familiar with basic concepts and have some experience with programming.

You should be familiar with:

  • Integrated Development Environments (IDE)
  • Java Developement Kits (JDK)
  • Basic concepts like loops, logic, variables

Variables

A variable in Java, is like a container or storage that holds a value in memory. It contains data for many types like numbers, texts, and boolean values.

There are certain rules to naming variables. Usual starts with a letter, and can follow with letters, digits, underscore. Camel Case is used, where the first word is lowercase and the first letter of the second word is uppercase, ex: firstName.

Declaring variables

In declaring a variable in Java, you must specify the type and give it a name, and assign a value.

type variableName = value;

Integer example:

int x = 3;

Double example:

double testScore = 90.5;

Boolean example:

boolean passTest = true;

String example

String message = "You passed the test!";
